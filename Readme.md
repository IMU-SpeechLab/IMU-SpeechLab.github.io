1. 首页：
首页部分对实验室工作方向进行介绍，实验室名字 IMU-SpeechLab。
IMU-SpeechLab 是内蒙古大学（IMU）计算机学院张学良教授领导的语音实验室，聚焦语音与音频信号处理及相关人工智能问题研究。我们旨在构建能够在复杂声学环境中稳健感知与理解语音和音频的机器聆听系统，重点关注混响条件下、多说话人并发与重叠语音等典型挑战。当前研究方向包括但不限于语音增强、说话人分离、目标说话人提取、语音去混响、主动降噪等，致力于推动相关语音算法在真实场景中的稳健应用。

首先是老师，包括照片和介绍：
张学良，教授，博士生导师，博士毕业于中国科学院自动化研究所模式识别重点实验室，大象声科联合创始人及CEO。内蒙古自治区自然科学二等奖获得者。主持国自然面上、国自然地区、内蒙古自治区科研基金项目、内蒙古大学高层次人才引进科研启动项目等多项国家/省部级项目。主要研究方向为语音增强，语音信号处理，计算听觉场景分析，语音合成，语音分离。
